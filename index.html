Sobre

Bem-vindo ao projeto <codAbili>, um programa educacional destinado a ensinar programação por meio da aprendisagem visual para pessoas neuroatipicas.

O objetivo deste projeto é fornecer uma forma de aprendisagem efetiva e inclusiva para aqueles que tem dificuldades com o método tradicional de ensino de programação. Sabemos que muitas pessoas neuroatipicas possuem uma maneira única de processar informações, e nosso programa foi desenvolvido para se adequar a essas necessidades.
